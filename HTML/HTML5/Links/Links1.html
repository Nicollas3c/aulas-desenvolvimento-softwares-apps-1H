Links

São utilizados para fazer a ligação entra duas páginas
Nesta página daremos três exemplos de links:

  1. No primeiro link vamos acessar uma página que está na mesma pasta
  2. O segundo link vai fazer uma ligação com uma página de outra pasta
  3. No último link vamos acessar uma página que pode estar em qualquer lugar na web

O link é criado com a tag "a" composta por:
  • a tag "a"
  • o endereço da página a sr acessada
  • o texto que será exibido na página
  • atenção a única coisa que é exibida do link na página é o texto
